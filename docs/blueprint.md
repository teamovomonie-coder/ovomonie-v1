# **App Name**: OVO Thrive

## Core Features:

- Customized Dashboard: Customized Dashboard: Display relevant financial information and quick actions based on user behavior.
- AI Assistant: AI Assistant: An intelligent, multilingual tool that provides real-time help and banking solutions via voice and text.
- MemoTransfer: MemoTransfer: Allows users to add photos and custom messages to transaction receipts for a personal touch.
- Custom ATM Card: Custom ATM Card: Enables users to personalize their ATM cards with photos or logos.
- Scan to Pay: Scan to Pay: Simplifies payments with QR code technology for instant and secure transactions.
- Bill Payments: Bill Payments: Consolidates bill payments for various services in one place, with recurring payment options.
- Custom Account Numbers: Account Number Customization: Use the last 10 digits of phone number as account number

## Style Guidelines:

- Primary color: Saturated teal (#008080) to represent reliability, trust and financial awareness, while avoiding looking generic
- Background color: Light desaturated teal (#E0F8F8).
- Accent color: Blue-green (#00CED1) to highlight key interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for headlines or body text.
- Use clean and modern icons related to banking and finance.
- Employ a clean, card-based layout with clear sectioning to enhance user experience.
- Subtle transitions and animations for loading screens and user interactions.
